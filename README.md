# Shourya's portfolio
## Description
This is my portfolio, I build this website to add to my slack page, LinkedIn, Instagram, and other apps that provide connections and outreach.
## Features
- Scroll animations
- Hidden animations everywhere
- eMail me straight from the site
- See some of my top projects
- Click on the cards to see my socials

## Tech Stack:
- Vite
- TypeScript
- CSS
- HTML
- Vercel
- React
- Node.js

### Future Plans
I plan on adding more features to this portfolio such as playing cards that you can hover on and they stick out of the deck that you can see my socials from there. I want this portfolio to be very unique and different from other ones.
